# C# To-Do App

A simple to-do application built with C# for managing backlog items.

## Overview

This application provides a straightforward interface for creating, viewing, and managing backlog items. It's designed to help users organize their tasks and track progress efficiently.

## Features

- Create new backlog items
- View all backlog items
- Mark items as completed
- Delete backlog items

## Getting Started

To get started with this application, clone the repository and build the solution using Visual Studio or the .NET CLI.

## Technologies Used

- C# (.NET Core)
- Console Application
